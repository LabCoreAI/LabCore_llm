# Changelog

All notable changes to this project are documented in this file.

The format is based on Keep a Changelog, and this project follows semantic versioning.

## [0.2.0] - 2026-02-27

### Added
- Modular GPT training stack with char and BPE tokenization paths.
- Data preparation pipeline for `txt/npy` and `bin/meta.json` formats.
- CLI training and text generation workflows.
- Gradio demo for local and Hugging Face inference.
- Export script for Hugging Face format and GGUF quantization helper.
- Instruction fine-tuning script (LoRA) for HF-compatible CausalLM models.
